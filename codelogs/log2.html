CSS Measuring Units

What's the difference between absolute and relative CSS length units?

The difference between relative and absolute CSS legnths are that absolute measurements represent a fixed enviromnent, unchangable environment. They don't scale based on zoom or screen size, it's always the same no matter what.

Relative units on the other hand change depending on what context is imminent, such as screen size, a parent element's size, and so on. So, the difference is that absolute is stationary while relative is adaptive.

Differences between certain CSS units

For the units I'm going to describe, I'll be using the following:

  • em, rem
  • vw, vh
  • %

em & rem

Em is the measurement of an element's parent. For example, if a page's font size is 12pt., then the em would be 1em = 12pt, 2em = 24pt, and so on. Its basically a measurement of a another, already-set measurement for that spot, like how you can change font size in a Google doc; it can change the font whjere it's placed.

Rem is the measurement is the font size of a root element rather than the parent. It plays the same as an em, with the 1 = x1 size, 2 = x2 size, however rem keeps the font size consistent, thus the 'root' instead of 'parent'.

Vw & vh

Vw and vh are measurements based on the viewport of the page, meaning that they measure the width/height of the webpage. So. 1vh is 1% of the webpage's height, while 1vw is 1% of the webpage's width. They're useful to scale browser windows, good for fullscreen sections and small sections, such as for a small 'feedback' window.

Percentage %

A percentage unit refers the size of something called the parent container, and is just a specific measurement on scaling selected items, such as margins, width, and font size. So say is you want a sentence or paragraph 50% larger, you just simply use "font-size: 150%;" and whatever other tweaks you wish. However if you want it 50% smaller, then you use "font-size: 50%;", being 50% of the parent's size rather than 50% larger.
