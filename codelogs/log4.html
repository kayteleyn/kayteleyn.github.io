CSS Colors

How does CSS use color?

CSS uses color through multiple ways, with some of which including:

  • background-color
  • border-color
  • color

These all play significant parts in coloring a website. Background-color, as you could figure, changes the entire background color to your desired color, by typing something like "background-color: fuschia;". The color tag is used to change the color of any text, and can be used with different lines of text when altering different elements. Lastly, border-color changes the color of a border arouind a certain element. However in order for the color to appear, you must also use the tag "border-style" and set it at either "solid" or "dashed".

How does CSS accept color as a value?

There are multiple ways in which CSS processes and accepts color, with some of them being:

  1. RGB

    This method is a way to precisely use red green and blue in your color, matching it exactly to your liking by putting numbers in order of red, green, blue.

    Example: "p.rgb-example { color: rgb(20, 32, 40); }".

  2. Hexadecimal

    HExadecimal is also a precise way of getting a desired color, however rather than putting the exact numbers, this one is a 6 character code, being something like "#rrf321". It's in CSS as: "p.hex-example { color: #rrr321; }".

  3. Named

    Named color format is a fast way to get a color on something, however its less precise than RGB and hexadecimal, since its a premade color rather than a specific code. It's the most simple, taking code as: "example { color: pink; }".

  4. HSL

    HSL format is a way to change the hue, saturation, and lightless, with those factors standing for HSL. This one is better for altering colors rather than setting them, with the CSS format being something like: "example { hsl(30, 50%, 60%); }". The hue is a basic number, while saturation and lightless use precentages.
